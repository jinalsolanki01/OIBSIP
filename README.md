# OIBSIP – Oasis Infobyte Data Analytics Internship

This repository contains all the tasks completed during the Oasis Infobyte Data Analytics Internship Program.

## Tasks Included

1. Exploratory Data Analysis (EDA) on Retail Sales Data
2. Customer Segmentation Analysis
3. Cleaning Data
4. Sentiment Analysis
5. Predicting House Prices with Linear Regression
6. Wine Quality Prediction
7. Fraud Detection
8. Unveiling the Android App Market: Analyzing Google Play Store Data
9. Autocomplete and Autocorrect Data Analytics 

## Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook
- Data Visualization
- Machine Learning Models
